### What I'm Working On

🐾 **AnimalDot** — Smart pet bed that passively monitors vitals (heart rate, respiration, temperature, weight) using geophone vibration sensing, DHT22, and FX29K load cells. Seven-module ESP32 firmware in PlatformIO, BedDot-compatible MQTT protocol, React Native/Expo companion app.

🖥️ **[Edusei Workstation](https://github.com/jke48222/Edusei-Workstation)** — Immersive 3D portfolio site with a retro CRT terminal, cinematic camera transitions, and interactive project exploration. React Three Fiber + Zustand + Framer Motion. → [jalenedusei.com](https://www.jalenedusei.com)

🗳️ **NSBE Live Election Platform** — Real-time chapter election system with live vote tallying and role-based access. Next.js 14 + Supabase.

🛰️ **MEMESat-1** — Flight software for a CubeSat mission using NASA's F Prime framework.

🎮 **Kitchen Chaos VR** — VR cooking game built in Unity for Meta Quest 3.

⚡ **FPGA Projects** — Segmented prime sieve engine (PrimeForge) and SD→DDR2→VGA sprite display pipeline (FPGAMEMVGA) on Nexys A7-100T in Verilog.
